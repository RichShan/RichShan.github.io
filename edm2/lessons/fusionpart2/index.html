Generating 2D Drawings

I used Kevin Kennedy's Youtube Video to learn how to generate 2D drawings in Fusion. To make a 2D drawing, I first navigated to the design I wanted to make a drawing of. I then clicked on the File dropdown menu, and clicked New Drawing -> From Design, making sure to select the object I wanted a drawing of. For practice creating 2D drawings, I used the Fusion360 example design for a connector joint.
